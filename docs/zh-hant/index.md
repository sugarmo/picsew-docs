# Picsew 文檔

歡迎來到 **Picsew** 官方文檔，這是一款功能強大的 iOS 截圖和網頁快照應用程序。

## 功能特色

Picsew 是一款綜合性的截圖和屏幕錄製應用，提供以下功能：

- **長截圖**：創建任何應用內容的滾動截圖
- **屏幕錄製**：錄製屏幕並自動生成長截圖
- **網頁快照**：直接從 Safari 捕獲完整網頁截圖
- **iOS 小組件**：通過 Today 小組件和控制中心快速訪問
- **設備模型**：為截圖添加專業的設備邊框
- **X-Callback-URL**：通過 URL 方案實現工作流自動化
